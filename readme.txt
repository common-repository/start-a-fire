== Description ==

Start A Fire lets you to add content recommendations within any external link you share. 

Using this plugin, you can add your content recommendations to any external link you share within your posts. For example, if you publish a post that contains a link to an article on The New York Times, you can use Start A Fire to add your content recommendations to that link.

This way, you can recommend relevant posts you wrote to your audience.

Example: [http://bit.ly/29Tbosn](http://bit.ly/29Tbosn) (look at the bottom right corner)

== Installation ==

Sign up for free on [Start A Fire](http://startafire.com/).

1. Upload the plugin folder to the `/wp-content/plugins/startafire` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Start A Fire screen to get your unique API key and connect your Start A Fire account.
4. Start adding your badge to links you share in your posts. You can also choose to add your badge to all links you previously created, before installing this plugin.
